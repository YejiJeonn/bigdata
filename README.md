# bigdata
[빅데이터] 수업용📊

### 주제 : 주가에 영향을 끼치는 다양한 금융상품 탐색을 위한 연관관계 및 비교 그래프 시각화 & 분석
이 레포지토리는 빅데이터 수업의 최종 과제 제출을 위해 생성되었습니다.
최근 주식에 관심이 생겨, 다양한 금융상품에 따라 주식의 변동성이 어떤 영향을 받는지 조사하게 되었고,
다양한 금융상품과 주식 간의 변동성 비교 그래프를 통해 어떤 금융상품에 어떠한 영향을 받는지에 대해 그래프로 시각화를 하고 분석하여
최종적으로 **금융상품에서 오는 영향으로 인한 주식의 변동성을 예측**하고자 했습니다.
